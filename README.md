# Tech News Recommender

A personalized tech news recommendation system that helps users discover relevant technology articles based on their interests.

## Features
- Personalized tech news recommendations
- Interest-based filtering
- Content-based recommendation engine
- Real-time tech news fetching
- Interactive user interface

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- NewsAPI

## Recommendation System
The platform uses content-based filtering with TF-IDF vectorization and cosine similarity to recommend relevant articles.
